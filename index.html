C#

C# é uma linguagem de programação, multiparadigma, de tipagem forte, desenvolvida pela Microsoft como parte da plataforma .NET. A sua sintaxe orientada a objetos foi baseada no C++ mas inclui muitas influências de outras linguagens de programação, como Object Pascal e, principalmente, Java.

Ruby

Ruby é uma linguagem de programação dinâmica e orientada a objetos, criada pelo japonês Yukihiro Matsumoto, em 1995. É uma linguagem popular, fácil de usar e aprender, e é usada para diversas finalidades, como o desenvolvimento de aplicativos web e análise de dados.

CSS

CSS é a sigla para Cascading Style Sheets, uma linguagem de marcação que define como os dados são apresentados em um navegador. O CSS é usado para estilizar elementos escritos em HTML, separando o conteúdo da representação visual do site.

HTML

HTML (HyperText Markup Language) é uma linguagem de marcação que serve para criar e estruturar páginas na internet. É um dos elementos mais básicos da web e é fundamental para quem quer desenvolver conteúdo para a internet.

JavaScript

JavaScript é uma linguagem de programação que permite criar interatividade em páginas web e aplicativos. É uma das linguagens mais populares do mundo e é considerada fundamental para programadores.

PHP

PHP é uma linguagem interpretada livre, usada originalmente apenas para o desenvolvimento de aplicações presentes e atuantes no lado do servidor, capazes de gerar conteúdo dinâmico na World Wide Web.

Python

Python é uma linguagem de programação de uso geral, multiplataforma, e de código aberto, que é usada para criar uma grande variedade de aplicações. É uma linguagem popular e versátil, conhecida por ser fácil de aprender e de usar, e por ter uma sintaxe clara.
